
Toy Server
==========

Gabriela Surita

Projeto desenvolvido para a disciplina de EA072 - Laboratório de Software Básico

Instalação
----------

Com GCC, Flex e Yacc instalados, execute:

```
make build
```

Testando
--------

Para visualizar um teste para um único exemplo:

```
make tests-once
```

Ou para executar todos os testes:

```
make tests
```
